## To run the project locally

### Frontend

Clone the repo locally
```
cd Twitter-clone-frontend
npm install
npm run start
```

### Backend

Clone the repo locally
```
git clone https://github.com/yashgangwar7558/Twitter-clone-backend
```
```
cd Twitter-clone-backend
npm install
npm run start
```
